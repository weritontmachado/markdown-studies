# markdown-studies
Project to test markdown formatting capabilities in readme.md

## Text formatting

### 1. Titles

# H1 text
## H2 text
### H3 text
#### H4 text
##### H5 text
###### H6 text

### 2. Style 

**Bold text**

__Bold text 2__

*Italic text*

~~StrikThrough text~~

**Bold and _Italic_ text**

***All Bold and Italic text***

> Block quote text
